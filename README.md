# BMW M4 Competition – Carbon Luxury Display  
### NZXT Kraken Elite 280

Luxury motorsport–inspired custom display project for **NZXT Kraken Elite 280**,  
designed around the visual language of **BMW M4 Competition (G82)**.

This project focuses on **elegance, restraint, and premium materials**:
carbon fiber texture, subtle M accents, and slow, refined animations.

## Design Philosophy
- Carbon fiber as primary material
- Minimal information (CPU temperature only)
- No aggressive colors or fast animations
- Inspired by BMW M luxury interiors

> Performance, crafted — not shouted.

---

## Features
- Carbon fiber luxury background
- BMW M branding (minimal & centered)
- Subtle animated M stripes
- Soft temperature warning (>80°C)
- Optimized for 480×480 NZXT Elite LCD

---

## Requirements
- NZXT Kraken Elite 280
- NZXT CAM software
- Custom Display mode enabled

---

## Installation
1. Open **NZXT CAM**
2. Go to **Kraken → Display**
3. Select **Custom**
4. Upload files from the `layouts/` folder
5. Assign **CPU Temperature** as the only sensor
6. Set brightness to **75–80%**

Detailed setup available in `docs/cam_setup.md`.
