# NZXT CAM Setup Guide  
## BMW M4 Competition – Carbon Luxury

This guide explains how to correctly configure the display in **NZXT CAM**
to preserve the luxury and minimal design intent.

---

## Recommended Files
Use ONLY the following files from the project:

- `layouts/layout_static_carbon.png`
- `assets/animations/m_stripes_carbon_flow.gif`
- `assets/animations/temp_warning_soft_red.gif` (optional)

---

## Display Configuration

### Display Mode
- Mode: **Custom**
- Resolution: **Automatic (480×480)**

### Background
- File: `layout_static_carbon.png`
- Scaling: Fit
- Brightness: **75–80%**

---

## Sensor Configuration
Use **only one sensor** for elegance.

- Sensor: **CPU Temperature**
- Unit: Celsius
- Color: White / Light Gray
- Size: Medium (not large)

> Avoid adding CPU load, GPU, RPM, or percentages.

---

## Animation Overlay
- Overlay GIF: `m_stripes_carbon_flow.gif`
- Loop: Enabled
- Animation speed: Default

---

## Temperature Warning (Optional)
- File: `temp_warning_soft_red.gif`
- Trigger threshold: **80°C**
- Purpose: Gentle visual alert, not aggressive warning

---

## Final Check
Before saving:
- No flashing colors
- No fast motion
- No unnecessary data
- Carbon texture clearly visible

If it feels “too much”, it is.
